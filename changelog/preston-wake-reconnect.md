The workspace UI (apps/system_interface) now reconnects its live-updates WebSocket after the user's machine wakes from sleep, the window regains focus, or the network comes back. Previously the SSH-tunneled socket died during sleep without firing `onclose`, so the UI silently showed pre-sleep state until a manual reload; it now tears down the phantom socket and opens a fresh connection whose replayed snapshot brings the view current, including rebuilding the pending proto-agent set so completed agent builds no longer show "Creating agent..." forever.
