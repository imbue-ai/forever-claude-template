The Caretaker is now a single, persistent agent invoked the same way every night, instead of a fresh agent created from scratch each time:

- There's now just one Caretaker skill. On its very first run it introduces itself with the welcome message and asks how you'd like it to help, then stops. On every later run it does the usual nightly routine -- greets you, checks your apps (with your permission), and tells you what it found or fixed.

- Each night your chat with the Caretaker is cleared before it checks in again, so it always starts fresh. It remembers your choices and what it's seen on previous nights through its own saved notes, not the conversation.
