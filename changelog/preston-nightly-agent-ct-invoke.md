The Caretaker is now one persistent assistant that stays with you, instead of a brand-new one being made every night:

- There is a single Caretaker tab that lives on across runs. Each night it simply clears its own chat and starts a fresh run in place, rather than being thrown away and recreated. If you'd closed its tab, it re-surfaces and gently flashes again so you notice the new run.

- Clearing the chat now actually works. When the Caretaker (or you) clears a conversation, the old transcript disappears and the chat starts empty -- previously a cleared chat would still show the prior conversation with the new one tacked on underneath.

- The Caretaker greets you the same way every night. The first night it welcomes you; later nights it opens with a short hello and says what it's about to do. The behind-the-scenes trigger that starts each run stays hidden from the chat, so the first thing you see is always the Caretaker speaking to you.
