- Activity indicator: a chat turn that runs longer than 20s now surfaces a "Skip" control (with an elapsed `m:ss` timer) in the activity strip above the composer. It is time-gated -- below the threshold the indicator behaves exactly as before -- and gives a "this is taking too long" escape hatch that is more discoverable than the always-on stop icon in the composer. Clicking Skip reuses the existing interrupt endpoint (`POST /api/agents/<id>/interrupt`), ending the in-progress turn and handing control back. The elapsed clock is per-agent and restarts on each new turn; the indicator ticks once a second only while a turn is in flight, so an idle chat stays quiet.
