
- Claude login modal: the "Open sign-in page" button and "Copy the link" fallback now produce a single, clean OAuth URL again. The `claude auth login` CLI started rendering the authorize URL as an OSC 8 terminal hyperlink wrapped in ANSI color, so the raw PTY stream carried the URL twice (once as the hyperlink target, once as the styled visible label) interleaved with escape sequences. The URL parser matched that stream verbatim, so the modal handed the user a doubled link with embedded escape codes. The parser now strips ANSI/OSC escape sequences from the consumed buffer before extracting the URL, collapsing the hyperlink back to one bare URL.
